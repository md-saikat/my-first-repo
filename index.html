Data Structure

Author: Md. Sezan Mahmud Saikat

Data structure is a way of organizing data inside of the computer main memory. It's one of the core topics in Computer science that every student should know.

Types of data structure
  • Linear data structure
  • Non-linear data structure
Linear Data Structure

A data structure will be called linear if its elements form a sequence. There are several linear datastructure types.

  • Array
  • Linked list
  • Stack
  • Queue
Non-linear Data Structure

A data structure will be called linear if its elements don't form a sequence. There are several linear data structure types.

  • Tree
  • Graph
